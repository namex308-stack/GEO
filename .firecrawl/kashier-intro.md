# Introduction

## Kashier Dashboard API

Integrate the [Dashboard API](https://developers.kashier.io/dashboard-api/api-basics) into your server applications for an automated experience without the need for manual dashboard operations. It allows you to create and share invoices, sync your inventory with product pages, process refunds, and utilize other Kashier Dashboard capabilities.

## Kashier Payments

Integrate secure payment processing capabilities into your web and mobile applications. Kashier offers [Payment Sessions](https://developers.kashier.io/payment/payment-sessions), [Hosted Checkout Page](https://developers.kashier.io/payment/payment-ui-builder) integration.

## 3rd Party Plugins

Easily plug and play with ready-made [plugins](https://developers.kashier.io/e-commerce/ecommerce-plugins). Kashier offers [WooCommerce](https://developers.kashier.io/e-commerce/woocommerce-payment), [Shopify](https://developers.kashier.io/e-commerce/shopify), [PrestaShop](https://developers.kashier.io/e-commerce/prestashop), [Magento](https://developers.kashier.io/e-commerce/magento), [OpenCart](https://developers.kashier.io/e-commerce/opencart), [Booking BA](https://ba-booking.com/shop/downloads/babe-payment-kashier), [CS-Cart](https://developers.kashier.io/e-commerce/cscart), and [VikBooking and VikAppointments](https://developers.kashier.io/e-commerce/vikwp).

You can use the Kashier APIs in test mode to test your integrations. When going live, remember to update your keys and URLs to live mode.
